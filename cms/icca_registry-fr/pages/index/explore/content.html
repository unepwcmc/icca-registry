Fournir une étude de cas est une partie facultative du processus d'inscription. Toutes les APAC qui sont dans la base de données de l'ICCA Registry n'ont pas une étude de cas sur cette page.

Vous pouvez utiliser la page Participer pour inscrire votre propre APAC ou soumettre une étude de cas. Ou en lire plus sur les bénéfices et les considérations relatifs à l'inscription.
